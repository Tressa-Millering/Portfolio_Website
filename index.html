Wandering in the Woods

Python Numpy MatPlotLib

The program first takes validated inputs for N (grid size, NxN), T (ticks per simulation), and R (number of simulations). It then runs R simulations. This entails simulating a grid with two people starting on opposite ends, each moving in a random direction each tick. If they were to move off the grid, they simply don't move at all. The simulation ends when, after moving, the two people are at the same grid location; alternatively, it will end when this loop has iterated T times without a meeting. Each simulation that ends in a meeting records how many ticks it took to get there and the coordinates of the meeting location. Finally, the program writes the maximum, minimum, and mean number of ticks over all simulations, a histogram of ticks per simulation, and a heat map of meeting locations. These results are then saved to a timestamped output file. View this project on GitHub
